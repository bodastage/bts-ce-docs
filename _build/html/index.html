Welcome to Boda Telecom Suite - CE¶

Release v1.0.0

Boda Telecom Suite - Community Edition (BTS-CE) is an open source telecommunication network management platform. The project aims to leverate the best in open source software , big data and artificial intelligence to provide a feature rich platform for network management. The project was started in 2017 by Emmanuel Robert Ssebaggala (founder of Bodastage Solutions ) with a vision of setting up a platform and building a community around it to foster innovation in the telecommunication domain. Initial focus is on Radio Network Optimization for networks with more than one vendors and technology.

An enterprise edition and hosted version are available at http://www.bodastage.com

Note

There are a number of topics that aren’t written yet but we continuously updating the documentation. Issues with the documentation should be reported at https://github.com/bodastage/bts-ce-docs/issues
